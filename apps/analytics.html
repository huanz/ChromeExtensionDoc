Analytics（分析）

描述： 使用 Chrome Platform Analytics JavaScript 库测量用户与您的应用的交互。
示例： JavaScript 和 Closure 的例子
了解更多： Chrome Platform Analytics

概述

Google Analytics（分析）平台 使您可以在不同的设备和环境下测量用户交互，这一平台可以收集、存储、处理和报告用户交互。Chrome Platform Analytics（CPA）是一个客户端库，使您可以在 Chrome 应用与扩展程序中收集用户交互信息，并将这些信息发送至 Google Analytics（分析）平台。

CPA 库遵循 Google Analytics（分析）的基本使用模式，并且对 Google Analytics（分析）的现有用户来说很熟悉。然而，与官方的以 Web 为中心的 Google Analytics（分析）JavaScript 库不同的是，CPA 使用一种“以应用为中心”的模型，更适合 Chrome 应用。此外，它还为特定匹配类型提供了专门的方法（例如“event”和“appView”），并且既可以在基于 Closure 的项目中使用，也可以在传统的 JavaScript 项目中使用。

隐私权

Google Analytics（分析）政策要求您向用户告知分析跟踪的使用，并提供停用跟踪的选项。CPA 库提供了库级别的支持使用户禁用跟踪，可以使您更容易地建立隐私权友好的应用。

库

CPA 托管在 GitHub 上：chrome-platform-analytics。

库文件为 google-analytics-bundle.js。
